Prework Study Guide

HTML

  • The head element contains information about the webpage and metadata for the browser.
    • The <meta> tag holds data for the browser
    • The <title> tag denotes the page title
  • The body element represents the visible content shown to the user.
    • The <h1> through <h6> tags denote headings
    • The <ul> and <ol> tags create unordered and ordered lists, with <li> being used to create list items

CSS

  • CSS or Cascading style sheets is a coding language which is used to dictate visual attributes for a document coded using another language
    • A margin indicates how much space we want around the outside of an element.
    • A padding indicates how much space we want around the content inside an element.
    • CSS files must be linked to an HTML document for changes to be applied

Git

  • Git is an online collaborative development platform which allows users to make iterative changes to programs by using "branches"
    • All git commands begin with "git"
    • git checkout is used to create and access branches in git
    • git status is used to fetch the current branch opened
    • git checkout -b branch-name: creates a new branch and switches to it
